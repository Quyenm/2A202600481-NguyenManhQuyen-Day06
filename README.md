# Vinmec Assistant

Vinmec Assistant is an AI-powered tool that helps patients prepare before visiting Vinmec by generating a personalized pre-visit checklist.

## Features
- fasting guidance
- required documents
- booking suggestion
- estimated arrival time
- nearest Vinmec facility support
